# Shor-s-Algorithm-Implementation-via-Qiskit
Este repositório traz uma implementação prática do Algoritmo de Shor utilizando Qiskit e a nuvem da IBM Quantum. O foco é demonstrar que computadores quânticos conseguem fatorar números inteiros rapidamente, provando a vulnerabilidade dos sistemas atuais de criptografia (como o RSA) frente a essa nova tecnologia.
